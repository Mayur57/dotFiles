# dotFiles
Essential files and preferences aggregated to make setting up a Mac as easy as running one script.

## Usage
Download the source as a ZIP, unpack and run `cd <your-dir> | sh setup.sh` in your terminal

## Compatibility
- MacOS 10.15+
